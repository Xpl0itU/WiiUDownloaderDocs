# How to install NUSspli

:::info GIF Guide
![NUSspli Setup](</gifs/NUSspli Setup.gif>)
:::
1.  Download NUSspli from the [latest release](https://github.com/V10lator/NUSspli/releases/latest).
    -   Make sure to download the right one: **Aroma** for Aroma, **Channel** for Tiramisu.
        ![NUSspli Setup](</screenshots/NUSspli.png>)
      :::warning
      Do not download any "DEBUG" files.
      :::
2.  Unzip the downloaded file by clicking **Extract All**,
    ![NUSspli Setup](</screenshots/NUSspliExtract1.png>)
    then click on **Extract**.
    ![NUSspli Setup](</screenshots/NUSspliExtract2.png>)
3.  Drag the `NUSspli` folder or `NUSspli.wuhb` to the `wiiu/apps/` directory on your SD card.
     ![NUSspli Setup](</screenshots/NUSspliDrag.png>)

Click [here](./use-on-wiiu.md) to return to the WiiU Guide.
:::info NUSspli vs. Lite
NUSspli Lite is just NUSspli without the ability to download apps directly on the Wii U.
NUSspli Lite has been discontinued in later versions. 
The latest version of NUSspli Lite is [here.](https://github.com/V10lator/NUSspli/releases/tag/v1.152)
:::
