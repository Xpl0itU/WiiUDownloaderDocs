# Installing WiiUDownloader on macOS
:::info Video Guide
<video src={require('@site/static/videos/MacOSGuide.mp4').default} controls preload style={{width: 100 + '%'}} playsinline autoPlay loop muted></video>   
:::
1.  Download `WiiUDownloader-macOS-Universal.dmg` from the [latest release](https://github.com/Xpl0itU/WiiUDownloader/releases/latest).
    ![DownloadMac](</screenshots/Mac1.png>)
2.  Go to your downloads and wait for it to finish.
    ![DownloadMac](</screenshots/Mac2.png>)
3.  Open the `.dmg` file.
    ![DownloadMac](</screenshots/Mac3.png>)
4.  Drag the **WiiUDownloader** icon to the **Applications** folder.
    ![DownloadMac](</screenshots/Mac4.png>)
5.  Open Launchpad and click on the **WiiUDownloader** icon.
    ![DownloadMac](</screenshots/Mac5.png>)
:::danger macOS Security
macOS will block WiiUDownloader because it is from an "unknown source". To bypass this, go to **System Preferences** → **Security & Privacy**, then to the right of "WiiUDownloader was blocked", click **Open Anyways** and it will open WiiUDownloader.
    ![DownloadMac](</screenshots/MacAnyways.png>)
:::

You have now completed the installation of WiiUDownloader on macOS.

Continue to [Using on a real Wii U](./use-on-wiiu.md) or [Using on Cemu](./use-on-cemu.md)
